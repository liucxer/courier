## CLI for courier


## Install

```bash
go get -u github.com/liucxer/courier/cmd/courier
```


## Usage

```
Usage:
  courier [command]

Available Commands:
  fmt         format and sort imports
  gen         generators
  help        Help about any command
  openapi     scan current project and generate openapi.json

Flags:
  -h, --help   help for courier
```
